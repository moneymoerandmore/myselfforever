`RelationshipGraph` 是数字分身的社会关系记忆层。

当前版本：

- [RelationshipGraph v0.1](RelationshipGraph.v0.1.md)
- [Schema](schema.md)

## 作用

它回答三个问题：

1. 这个人是谁？
2. 我和 TA 是什么真实关系？
3. 我平时应该如何和 TA 沟通？
