# SelfCore

`SelfCore` 是数字分身的核心人格层，用来描述“我”的稳定三观、思维方式、表达方式和边界。

当前版本：

- [SelfCore v0.1](SelfCore.v0.1.md)
- [Schema](schema.md)
- [评估用例](../../evals/cases/selfcore-v0.1.md)

## 版本原则

- 慢变量：三观、价值判断、核心心智模型。必须有强证据或用户确认才更新。
- 中变量：表达偏好、常用拆解路径、关系沟通规则。需要多次样本或校对反馈更新。
- 快变量：近期关注、新闻观点、当天情绪、临时目标。可以进入运行时记忆，不直接改写核心人格。

## 数据边界

本目录只保存结构化蒸馏结果，不保存全量聊天原文。
